Web Development

1. HTML

  • Day 1:

    "We started with the installation of VS Code on our laptops. Then, we learned different tags like the head tag, title tag, paragraph tag, horizontal line tag, break tag, and how to add comments to a page. The session ended with a summary."


  • Day 2:

    "We learned about heading tags (h1 to h6), anchor tag, image tag, bold, italic, underline, sub, sup, big, small, and pre tags. We also explored page layout techniques like header, main, and footer. Inside the main tag, we covered section, article, and aside tags. The session concluded with a summary and website presentations by our colleagues."


  • Day 3:

    "We learned how to create lists in a webpage, different types of lists, and how to add tables with attributes. We were also introduced to forms using HTML tags such as form, input, label, class and id, checkboxes, and more. The session concluded with a summary of all HTML lessons."


2. CSS

  • Day 4:

    "After the completion of HTML, we started with basic CSS in which we got to know about types of CSS (Inline, Internal, and External) and some basic attributes in it like height, width, text color, background color, font size, etc."


  • Day 5:

    "On the 5th day, we learned how to create a box with different attributes such as width, height, margin, border, etc. We also learned about display flex and its properties like position, justification, color, and alignment."


  • Day 6:

    "We created our portfolio, learned how to host it, and also got to know about different websites like GitHub and Claude.ai."
